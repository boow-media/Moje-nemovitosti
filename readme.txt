Plugin pro realitní makléře – přidávání, správa a zobrazení nemovitostí. Plná integrace s Meta Box.

== Description ==

Makléř+ je jednoduchý, ale výkonný plugin pro správu realitních zakázek:
- Vlastní post type „Nemovitosti“
- Uživatelsky přívětivá pole přes Meta Box
- Automatické formátování cen, ploch a dalších údajů
- Přehled v administraci (sloupec „Stav nabídky“)
- Podpora pro Breakdance Builder i další
- Automatické aktualizace přes GitHub

== Installation ==

1. Nahrajte složku `makler-plus` do adresáře `/wp-content/plugins/`
2. Aktivujte plugin v sekci „Pluginy“ ve WordPressu
3. Ujistěte se, že máte aktivní plugin [Meta Box](https://wordpress.org/plugins/meta-box/)
4. Vytvořte novou nemovitost v administraci

== Frequently Asked Questions ==

= Potřebuji Meta Box? =
Ano. Plugin využívá Meta Box pro zobrazení vlastních polí. Bez něj nebude fungovat.

= Funguje i s Breakdance nebo jinými buildery? =
Ano. Otestováno s Breakdance, Elementor i klasickým editorem.
